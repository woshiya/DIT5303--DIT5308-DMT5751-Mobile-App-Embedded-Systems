# MedBox: Medicine Reminder App and IoT Medicine Box

## Overview
This project aims to develop a mobile application and an IoT-enabled medicine box that assists patients, especially those with chronic conditions, in managing their medication schedules effectively. The solution reduces the reliance on memory and enhances the overall medication adherence experience.

# Project Description
The application integrates Bluetooth technology with an Arduino-based medicine box to send reminders to users when medication is due. It also logs usage data that healthcare providers can access for more accurate patient follow-ups
